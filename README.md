This project contains a virtual machine intended to provide first year students with an isolated environment to experiment freely with Linux and Java.

* Ubuntu 20.04.1 LTS
* gdm3, gnome-shell, gnome-terminal
* Firefox
* openjdk-14-jdk
* gedit,vim,nano
